# Runtime vehicle assets

The five GLBs in this directory are required startup assets generated by `npm run vehicle:build`:

- `citi-golf.glb`
- `vrrr-phaa-gti.glb`
- `hilux-bakkie.glb`
- `jmpd-interceptor.glb`
- `quantum-express.glb`

Do not edit exported GLBs directly. The editable Blender sources, committed recipes, previews, attribution, and checksum lock live under `art/vehicles/`. Runtime validation requires +Z forward, +Y up, tyre-contact grounding, named wheel/lamp/cabin nodes, opaque PBR materials, and contracted gameplay bounds.
